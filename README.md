* This mod was split from [Ecliptic Seasons](https://legacy.curseforge.com/minecraft/mc-mods/ecliptic-seasons). Because the Voxy port cannot be publicly released, its stability is poor.
  ![Voxy show](https://media.forgecdn.net/attachments/1640/684/2026-01-24_21.jpg)

  This mod was created to allow players to choose a suitable version of the Voxy port and compatible plugins.
